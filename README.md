# FLAMSENTINELLE sur Streamlit

Cette version permet de déployer ton application HTML sur Streamlit Community Cloud.

## Fichiers
- `app.py`
- `FLAMSENTINELLE.html`
- `requirements.txt`

## Déploiement
1. Crée un dépôt GitHub.
2. Envoie ces trois fichiers dans le dépôt.
3. Va sur Streamlit Community Cloud.
4. Clique sur **Create app**.
5. Sélectionne le repo, la branche, puis `app.py`.
6. Déploie.

## Important
Le mot de passe de l'écran de connexion est dans le HTML lui-même.
Ce n'est donc pas une vraie sécurité serveur.
